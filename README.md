## ✨ Phase 1 ফিচার

- ✅ Project Management (রঙ + ট্যাগ)
- ✅ Idea Tracking — todo / doing / done
- ✅ Priority — কম / মধ্যম / জরুরি
- ✅ File Manager — js, jsx, py, dart, html, css, json, yml, md...
- ✅ Code Editor — unlimited character
- ✅ Auto Versioning — v1, v2, v3...
- ✅ File Share, Copy, Delete
- ✅ SQLite local storage (offline)

## 🚀 GitHub-এ Push করার নিয়ম

```bash
git init
git add .
git commit -m "🚀 My Manager Flutter - Phase 1"
git branch -M main
git remote add origin https://github.com/hanif4343/my-manager-flutter.git
git push -u origin main
```

Push করলেই GitHub Actions APK build করবে!
**Releases** tab-এ APK পাবে।
